# Campus-Connect

CampusConnect — A smart campus engagement platform that personalizes event discovery, streamlines event management, and connects students with opportunities that match their interests


1. What is this project?

CampusConnect is a modern event management platform designed for colleges and universities.

It helps students discover relevant events, allows organizers to publish and manage events, and enables administrators to approve and monitor campus activities.

2. Why did I build it?

Many colleges rely on WhatsApp groups, posters, or scattered announcements for event communication.

CampusConnect aims to centralize event discovery and management while providing personalized event recommendations.
